# Acceso Manual al Panel Administrativo

## URL Directa
El acceso al panel administrativo está disponible únicamente mediante URL directa:

```
https://intevopedi.org/admin/login
```

## Credenciales
- Usuario: No requerido
- Contraseña: La configurada en la variable de entorno `ADMIN_ACCESS_PASSWORD`

## Seguridad
- El acceso admin **no** aparece en la navegación pública
- Solo es accesible mediante URL directa
- La contraseña debe ser segura y cambiarse regularmente

## Funciones Disponibles
Una vez autenticado, el administrador puede:
- Ver todas las inscripciones
- Actualizar estados de inscripción
- Gestionar pagos y asistencia
- Emitir certificados
- Ver estadísticas del campus

## Notas
- Mantener la URL privada
- No compartir la contraseña de administración
- El acceso se cierra automáticamente tras inactividad
